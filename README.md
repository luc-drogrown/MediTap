# MediTap

Folder structure:
-frontend in /src/client(UI, forms, etc.)
-backend in /src/server(API, DB manager, etc.)
-Automated tests, etc. in /tests
-documentation in /docs
-solution in main folder

Git instructions:
-always branch of main
-when implementing a feature create branch: /feat/your-feature
-when fixing an issue create branch: br/fix/issue-fixed
-a branch is merged after it has been reviewed and tested
