🌿 Git Instructions
Always branch off main
Naming Conventions

When implementing a feature:

When fixing an issue:

br/fix/issue-fixed
✅ Merge Policy
A branch is merged only after:
It has been reviewed
It has been tested

Keep branches focused on a single feature or fix
